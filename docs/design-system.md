# 闪学设计系统

## 1. 风格方向

采用与 Gizmo 同类型的扁平化、简约、轻游戏化学习工具风格：白底、清晰排版、鲜明但克制的行动色、简单图形、低阴影、低装饰。

不得复制 Gizmo 的品牌、角色、插图、图标或具体页面布局。闪学需要形成自己的中文品牌：快、清楚、可信、适合考试复习。

## 2. 视觉关键词

- 清爽
- 快速
- 低压力
- 游戏化但不幼稚
- 可信赖
- 中文长文本友好

## 3. 色彩

- 背景：`#FFFFFF`、`#F7F8FA`
- 主文字：`#17202A`
- 次文字：`#667085`
- 主行动蓝：`#2F80ED`
- 学习绿色：`#22C55E`
- 提醒黄色：`#FACC15`
- 错误红：`#EF4444`
- 分割线：`#E5E7EB`

绿色只用于掌握、关键高亮和正确反馈；蓝色用于主要按钮；黄色用于成就、连胜和提醒。

## 4. 字体与排版

- 中文优先使用系统字体：PingFang SC、Microsoft YaHei、HarmonyOS Sans。
- 题干正文 16-18px，解释 15-16px。
- 卡片标题不超过 22px。
- 不使用负字距，不按视口宽度缩放字号。
- 所有按钮和卡片必须支持中文长词、英文术语和公式换行。

## 5. 组件

- 卡片：8px 圆角以内，轻边框，不做厚重阴影。
- 按钮：主按钮蓝底白字，次按钮白底边框，图标按钮必须有 tooltip 或辅助文本。
- 进度：掌握率用细进度条或环形小图，不做大面积炫彩。
- 题目选项：固定高度下限，选中态清晰，答错后显示红色边框和解释入口。
- 高亮词：绿色浅底，不遮挡原文。
- 成就：小徽章，不占据学习主屏。

## 6. 关键页面

### 首页

优先展示今日复习、继续学习和快速导入。避免营销首页和大图 hero。

### 导入页

提供 4 个清晰入口：拍照、文件、文本、链接。导入过程中显示进度和可取消。

### 卡片审阅页

左侧/上方显示卡片列表，主区域编辑当前卡片。移动端使用单列，底部固定保存按钮。

### 测验页

题干居中但不夸张，选项可单手点击。解释在答题后展开，默认显示简短解释和来源入口。

### 小组页

突出今日挑战、成员进度、共享卡组。排行榜只展示学习行为，不展示敏感个人信息。
